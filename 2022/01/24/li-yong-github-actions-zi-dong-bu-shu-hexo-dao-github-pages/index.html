利用 Github Actions 自动部署 Hexo 搭建博客到Github Pages

Github Github Actions 博客
教程
发布日期:   2022-01-24

开始之前准备

本地安装

  • Node.js
  • Git

所需账号

  • Github账号

请自行下载安装配置环境和注册账号

开始

部署Hexo

一、打开Git Bash Here

桌面右键点击打开

按照Hexo官网命令复制粘贴

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

输完最后一个命令出现下图表示成功

然后用浏览器打开 http://localhost:4000/ 可以看到部署的hexo博客

Ctrl+C停止程序

需要更换主题，自行更换并配置

如过程中出现错误先点击此处查找解决访问，然后再行提问

配置

  • 打开桌面blog文件夹

  • 打开_config.yml文件，配置deploy:

deploy:
  type: git
  repository: git@github.com:你的用户名/你的用户名.github.io.git
  branch: main

创建workflows

  • 在blog下创建文件夹.github\workflows新建文件deploy.yml
  • workflows下新建文件deploy.yml文件名字可以自己改，但是文件一定要放在 .github/workflows 目录中
  • 文件中内容如下
    # Actions workflows name
name: HEXO DEPLOY

# 触发条件
on:
  workflow_dispatch:

# 环境变量
env:
  GIT_USER: ${{secrets.GIT_USER}}
  GIT_EMAIL: ${{secrets.GIT_EMAIL}}

jobs:
  build: 
    runs-on: ubuntu-latest 
        
    steps:
    # check it to your workflow can access it
    # from: https://github.com/actions/checkout
    - name: Checkout
      uses: actions/checkout@main
      
    # from: https://github.com/actions/setup-node  
    - name: Setup Node.js
      uses: actions/setup-node@main
      with:
        node-version: "16.x"
    
    - name: Setup Hexo Dependencies
      run: |
        npm install hexo-cli -g
        npm install
    
    - name: Setup Deploy Private Key
      env:
        HEXO_DEPLOY: ${{secrets.HEXO_DEPLOY}}
      run: |
        mkdir -p ~/.ssh/
        echo "$HEXO_DEPLOY" > ~/.ssh/id_rsa 
        chmod 600 ~/.ssh/id_rsa
        ssh-keyscan github.com >> ~/.ssh/known_hosts
        
    - name: Setup Git Infomation
      run: | 
        git config --global user.name $GIT_USER
        git config --global user.email $GIT_EMAIL
    - name: Deploy Hexo 
      run: |
        hexo clean
        hexo deploy

Github方面

登录GitHub

创建仓库

  • 一个私有仓库

私有仓库名字可以随便起

  • 一个公共仓库

公共仓库名必须为：你的用户名.github.io

生成秘钥

打开Git Bash Here输入:

ssh-keygen -t rsa -C "email@example.com"

一直回车到结束
会生成两个文件，一个私钥一个公钥
默认生成到C:\Users\用户名.ssh里

配置公钥

进入github设置

SSH and GPG key -> 新建密钥New SSH key

打开id_rsa.pub
把里面内容复制粘贴到key里面，然后添加

添加完后测试一下
打开Git Bash Here输入:

ssh -T git@github.com

首次需要同意
当出现下图表示成功

配置私钥-机密

进入私有仓库 -> Settings -> Secrets

创建三个secrets

  • GIT_EMAIL：填入你的GitHubEmail，保存
  • GIT_USER： 填入你的GitHub用户名，保存
  • HEXO_DEPLOY：填入私钥即打开id_rsa里面的内容，保存

把项目推送到私有仓库

打开桌面blog文件夹
输入命令：

hexo clean
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:你的用户名/你的私有仓库名.git
git push -u origin main

至此项目就推送到你的私有仓库

触发Workflow

进入私有仓库 -> Actions -> HEXO DEPLOY -> Run workflow

等待运行完成，就会生成推送到你的Github Pages上，即你的新建的共有仓库

打开网址，你的用户名.github.io，就会看到生成的博客了

至此已全部部署完成

后期添加文章只需要在/source/_posts/目录下添加编辑即可,编辑完成后触发一下这个Workflow就会自动更新到你的博客上
